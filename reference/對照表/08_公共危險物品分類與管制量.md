# 對照表 08：公共危險物品分類與管制量

> 依據：《公共危險物品及可燃性高壓氣體製造儲存處理場所設置標準暨安全管理辦法》§3、附表一
> ⚠️ 本表為法規快照之整理，**引用前請依官方現行版核對時效**；附表一完整權威版見 `statutes/附件/…/附表一…ODT`。

## 一、六類公共危險物品總覽（§3）

| 類別 | 性質 | 代表物質 | 管制量（要點） |
|------|------|----------|----------------|
| 第一類 | **氧化性固體** | 氯酸鹽、過氯酸鹽、無機過氧化物、硝酸鹽 | 第一級 **50 kg**／第二級 300 kg／第三級 1000 kg |
| 第二類 | **易燃固體** | 硫化磷、赤磷、硫磺（100 kg）；鐵粉（500 kg）；金屬粉、鎂（一級 100／二級 500 kg）；固態酒精（1000 kg） | 100～1000 kg |
| 第三類 | **發火性液體、發火性固體及禁水性物質** | 鉀、鈉、烷基鋁、烷基鋰（**10 kg**）；黃磷（**20 kg**）；金屬氫化物等（一級 10／二級 50／三級 300 kg） | 10～300 kg |
| 第四類 | **易燃液體及可燃液體** | 汽油、酒精、柴油、重油、動植物油 | 見下表（公升） |
| 第五類 | **自反應物質及有機過氧化物** | 有機過氧化物、硝酸酯類、硝基化合物 | A 型 **10 kg**／C 型 100 kg（B、D 型見原檔） |
| 第六類 | **氧化性液體** | 過氯酸、過氧化氫、硝酸 | 第一級 **300 kg**（第二級見原檔） |

**記憶點**：最嚴的是第三類鉀鈉烷基鋁鋰與第五類 A 型（都 10 kg）；黃磷 20 kg；一類一級 50 kg、六類一級 300 kg。

## 二、第四類（易燃液體及可燃液體）管制量——最常考

| 種類 | 定義（閃火點） | 分級 | 管制量 |
|------|----------------|------|--------|
| 特殊易燃物 | 自燃溫度 ≤100 ℃，或閃火點 <−20 ℃ 且沸點 ≤40 ℃ | — | **50 L** |
| 第一石油類 | <21 ℃（例：汽油） | 非水溶性 | **200 L** |
| 第一石油類 | | 水溶性 | 400 L |
| 酒精類 | C1～C3 一個飽和羥基 | — | **400 L** |
| 第二石油類 | 21～<70 ℃（例：柴油、煤油） | 非水溶性 | **1000 L** |
| 第二石油類 | | 水溶性 | 2000 L |
| 第三石油類 | 70～<200 ℃（例：重油） | 非水溶性 | **2000 L** |
| 第三石油類 | | 水溶性 | 4000 L |
| 第四石油類 | 200～<250 ℃（例：齒輪油） | — | **6000 L** |
| 動植物油類 | 閃火點 <250 ℃ | — | **10000 L** |

**記憶點**：50→200→400→1000→2000→6000→10000（特殊、一石、酒精、二石、三石、四石、動植物）；**水溶性一律非水溶性的 2 倍**。
易燃液體＝閃火點 ≤93 ℃；可燃液體＝閃火點 >93 ℃ 未滿 250 ℃。

## 三、多種物品合併計算（附表一附註）

儲存二種以上時：$\sum_i \dfrac{\text{各物品數量}_i}{\text{各物品管制量}_i} \geq 1$ 即視為達管制量。

## 四、常考倍數門檻（本辦法各條）

| 門檻 | 意義 |
|------|------|
| 管制量 **15 倍**未達 | 第一種販賣場所 |
| **15～<40 倍** | 第二種販賣場所 |
| **10 倍**以上 | 製造／一般處理場所四周保留空地由 3 m 加為 **5 m**；顯著滅火困難場所判定之一（詳設置標準 §194） |
| **30 倍**以上 | 消防法 §15-6：應遴用**保安監督人**訂定消防防災計畫 |
| LPG 串接使用 **80 kg** 以上 | 容器串接使用場所 |

## 五、可燃性高壓氣體（本辦法定義）

| 氣體 | 條件 |
|------|------|
| 壓縮氣體（氫、乙烯、甲烷、乙烷） | 常用溫度或 35 ℃ 時表壓力 ≥**10 kgf/cm²**（1 MPa） |
| 壓縮乙炔氣 | 常用溫度或 **15 ℃** 時表壓力 ≥**2 kgf/cm²**（0.2 MPa） |
| 液化氣體（丙烷、丁烷、LPG） | 常用溫度或 35 ℃ 以下時表壓力 ≥**2 kgf/cm²** |
