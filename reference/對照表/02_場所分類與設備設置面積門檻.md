# 對照表 02：場所分類與設備設置面積門檻

> 依據：《各類場所消防安全設備設置標準》§12（場所分類）、§15～§19（主要設備應設門檻）

## 一、場所分類（§12，甲乙丙丁戊）

| 類 | 性質 | 各目要點 |
|----|------|----------|
| **甲類**（7 目） | 不特定多數人、避難弱者，管制最嚴 | （一）戲院歌舞廳酒吧 KTV 等；（二）保齡球館集會堂健身遊藝場等；（三）觀光旅館飯店旅館（有寢室客房）；（四）商場市場百貨；（五）餐廳咖啡廳；（六）**醫院療養院長照老幼身障機構特殊學校**；（七）三溫暖公共浴室 |
| **乙類**（12 目） | 特定使用者為主 | （一）車站機場；（二）金融機構；（三）學校教室補習班；（四）圖書館博物館；（五）寺廟教堂納骨塔；（六）辦公室診所靶場；（七）**集合住宅、寄宿舍**；（八）體育館活動中心；（九）室內溜冰游泳池；（十）電影攝影場；（十一）**倉庫**傢俱展售場；（十二）**幼兒園** |
| **丙類**（3 目） | 機械／車輛 | （一）電信機器室；（二）汽車修護廠飛機庫；（三）室內停車場（空間） |
| **丁類**（3 目） | 工作場所 | 高度／中度／低度危險工作場所 |
| **戊類**（3 目） | 複合與地下 | （一）複合用途建築物**含甲類**用途；（二）其他複合用途建築物；（三）**地下建築物** |

**記憶點**：甲類七目口訣「娛樂（一二）、住宿（三）、商餐（四五）、醫養（六）、浴（七）」；乙類記特例：七目集合住宅、十一目倉庫、十二目幼兒園。

## 二、室內消防栓應設門檻（§15）

| 建築物 | 甲類（一）目 | 其他甲類目及乙丙丁類 | 學校教室 |
|--------|:---:|:---:|:---:|
| 5 層以下（任一層） | ≥**300 m²** | ≥**500 m²** | ≥1400 m² |
| 6 層以上（任一層） | ≥**150 m²**（各款均同） | 同左 | 同左 |
| 地下層／無開口樓層 | ≥**100 m²** | ≥**150 m²** | — |
| 地下建築物（總樓地板） | ≥**150 m²** | 同左 | 同左 |

免設：有效範圍內設有室外栓、撒水（含補助撒水栓）、水霧泡沫 CO₂ 惰性鹵化烴乾粉者；室外栓僅限第一層（水平 40 m）、第二層（步行 40 m）免設。

## 三、室外消防栓應設門檻（§16，一二層樓地板面積合計）

| 危險程度 | 面積門檻 |
|----------|:---:|
| 高度危險工作場所 | ≥**3000 m²** |
| 中度危險工作場所 | ≥**5000 m²** |
| 低度危險工作場所 | ≥**10000 m²** |

混合時：$\sum \dfrac{\text{實際面積}}{\text{各款門檻}} > 1$ 即應設。**記憶點**：高中低「3、5、10」千。

## 四、自動撒水應設門檻（§17）

| 情形 | 門檻 |
|------|------|
| 10 層以下——甲類（一）目 | 樓地板合計 ≥**300 m²** |
| 10 層以下——其他甲類目及乙類（一）目 | ≥**1500 m²** |
| **11 層以上之樓層** | ≥**100 m²**；供甲類或戊類（一）使用者全設 |
| 地下層／無開口樓層（甲類） | ≥**1000 m²** |
| 複合用途建築物（戊一）中甲類合計 | ≥**3000 m²** 時該樓層 |
| 高架儲存倉庫（乙十一） | 高度 >**10 m** 且 ≥**700 m²** |
| 地下建築物（總樓地板） | ≥**1000 m²** |
| **高層建築物** | 全設 |
| 長照、護理之家等避難弱者機構（§17Ⅰ九） | 一律應設；合計 <1000 m² 得改設**水道連結型**自動撒水設備 |

免設：有效範圍內設有水霧、泡沫、CO₂、惰性氣體、鹵化烴、乾粉者。

## 五、火警自動警報設備應設門檻（§19）

| 建築物 | 門檻 |
|--------|------|
| 5 層以下——甲類及乙類（十二）幼兒園 | 任一層 ≥**300 m²** |
| 5 層以下——乙（除十二）丙丁類 | 任一層 ≥**500 m²** |
| 6～10 層 | 任一層 ≥**300 m²**（不分用途） |
| **11 層以上** | 一律應設 |
| 地下層／無開口——甲（一）（五）目及含該用途之戊（一） | ≥**100 m²** |
| 地下層／無開口——其他 | ≥**300 m²** |
| 複合用途（戊一） | 總樓地板 ≥500 m² 且甲類合計 ≥300 m² |
| 甲類及地下建築物（戊三） | 總樓地板 ≥**300 m²** |
| 避難弱者機構（§19Ⅰ七） | 一律應設 |

免設：設有自動撒水、水霧或泡沫（限標示溫度 ≤75 ℃、動作時間 ≤60 秒之密閉型撒水頭）之有效範圍——但**甲類、地下建築物、高層建築物、應設偵煙式處不得免**。

## 常考比較

1. 「300／500」門檻組合出現兩次：室內栓（5 樓以下甲一 300／其他 500）與火警（5 樓以下甲類 300／乙丙丁 500）——注意火警的 300 涵蓋**全部甲類＋幼兒園**，室內栓的 300 僅**甲類（一）目**。
2. 11 層以上：火警**無條件**應設；撒水為 ≥100 m² 之樓層（甲類、戊一則全設）。
3. 避難弱者機構（榮家、長照、護理之家等）：撒水（§17Ⅰ九）與火警（§19Ⅰ七）皆一律應設，是近年修法重點。
