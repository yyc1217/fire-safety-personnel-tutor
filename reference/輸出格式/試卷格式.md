# 消防設備師／士 試卷格式範本

> 本檔擷取自 115 年（最新年度）消防設備人員考試各科**原卷 PDF**（`corpus/pdf/{師,士}/115/`）之版面格式，供 `/出考卷` 產生具高度相似性之模擬試卷使用。
>
> **⚠️ 本範本僅供模擬練習**：為明確與正式試卷區隔，凡依本範本產出之試卷，**第一行與最後一行一律加註 `【模擬考】`**（如本檔首尾所示）。正式考選部試卷不含此標記。

---

## 一、格式來源與版本

- 取樣年度：**民國 115 年**（`corpus/pdf/師/115/`、`corpus/pdf/士/115/`）
- 涵蓋兩種卷型：
  - **混合卷**（申論 50 分＋測驗 50 分）：師之消防法規、士之全部 4 科（含火災學概要）——範本：`0802_消防法規.pdf`
  - **純申論卷**（申論 100 分）：師之火災學及各系統設備科——範本：`0805_水系統消防安全設備.pdf`
- 師（高考）與士（普考）差異僅在表頭之「等別／類科／科目／考試時間／代號／頁次」欄位。

---

## 二、版面結構（逐區塊）

### 區塊 A — 表頭（試題卷首，師士共用）

原卷卷首為當年度綜合性考試全名（多科合辦，四行），照抄如下（**年度依需求替換**）：

```
{年度}年專門職業及技術人員高等考試大地工程技師考試
分階段考試（第一階段考試）、驗船師、第一次食品技師考試、
高等暨普通考試消防設備人員考試、普通考試地政士、專責
報關人員、保險代理人保險經紀人及保險公證人考試試題
```

接續考科資訊欄（冒號對齊，逐行）：

```
等　別：{高等考試｜普通考試}
類　科：{消防設備師｜消防設備士}
科　目：{科目名稱}
考試時間：{2 小時｜1 小時 30 分}
座號：
※注意：{禁止使用電子計算器｜可以使用電子計算器}。
代號：{5 位數，如 40120}
頁次：{總頁數}－{頁碼}
```

欄位對照（依等別／科別）：

| 欄位 | 消防設備師（高考） | 消防設備士（普考） |
|------|-------------------|-------------------|
| 等別 | 高等考試 | 普通考試 |
| 類科 | 消防設備師 | 消防設備士 |
| 考試時間 | 2 小時 | 1 小時 30 分 |
| 科目 | 火災學／消防法規／警報系統／避難系統／水系統／化學系統消防安全設備 | 火災學概要／消防法規概要／警報與避難系統／水與化學系統消防安全設備概要 |
| 卷型 | 消防法規為混合卷；火災學及四系統科為純申論卷 | 全部為混合卷 |

> `座號`、`代號`、`頁次` 為考場填寫／印製資訊，模擬卷可保留欄位樣式並以占位符呈現。

---

### 區塊 B — 申論題部分

混合卷以標題起始；純申論卷則省略標題（直接接作答須知＋題目）：

```
甲、申論題部分：（50 分）
```

作答須知（原卷固定三行，照抄）：

```
不必抄題，作答時請將試題題號及答案依照順序寫在申論試卷上，於本試題上作答者，不予計分。
請以黑色鋼筆或原子筆在申論試卷上作答。
本科目除專門名詞或數理公式外，應使用本國文字作答。
```

題目格式（國字數字題號；子題以全形括號序號分項並各自標配分）：

```
一、{題幹敘述，常以情境案例開場}
　　（一）{子題}（{配分} 分）
　　（二）{子題}（{配分} 分）

二、{題幹敘述}
　　（一）{子題}（{配分} 分）
　　（二）{子題}（{配分} 分）
```

- 混合卷申論通常 **2 題、每題 25 分**（子題配分加總為 25）。
- 純申論卷通常 **4 題、每題 25 分**，計算題會附「已知設計條件」與圖表（如水錘吸收器、摩擦損失水頭表）。
- 計算／公式一律 LaTeX（`$...$`、`$$...$$`），依 CLAUDE.md 公式格式規範。

---

### 區塊 C — 測驗題部分（僅混合卷）

區塊標題與另一組代號（測驗式試題代號與申論不同）：

```
乙、測驗題部分：（50 分）
代號：{測驗式代號，如 2401}
```

作答須知（原卷固定兩行，照抄）：

```
本試題為單一選擇題，請選出一個正確或最適當答案。
共 {題數} 題，每題 {配分} 分，須用 2B 鉛筆在試卡上依題號清楚劃記，於本試題或申論試卷上作答者，不予計分。
```

- 師：**40 題、每題 1.25 分**；士：題數依科目（概要科多為 40 題、每題 1.25 分）。

單題格式（阿拉伯數字題號獨立成行；四個選項 (A)–(D)）：

```
{題號}　{題幹}
(A) {選項}
(B) {選項}
(C) {選項}
(D) {選項}
```

- 題幹常見句型：「下列敘述何者**正確**？」「下列何者**錯誤**？」，或以情境（某場所／某設備竣工查驗）帶入。
- 選項多以四個平行敘述構成，數值型題目以單一數字或 `X＝…，Y＝…` 呈現。
- 部分題目以圈號子項組合作答：`①…②…③…④…`，選項為 `①②`、`①②④` 等組合。

## 三、模擬卷產出規範（`/出考卷` 遵循）

1. **首尾標記**：輸出之試卷**第一行與最後一行**各單獨加一行 `【模擬考】`，確保與正式卷可辨識區隔。
2. **忠於版面**：表頭欄位、作答須知、題號與選項格式一律比照本範本；年度、代號、頁次、座號以占位符或指定值填入。
3. **題型配比**：依科目自動選擇卷型——師之消防法規及士全部科目用混合卷，師之火災學及四系統科用純申論卷；申論與測驗之題數、配分依區塊 B、C 之慣例。
4. **命題來源與誠實性**：題目取材以 `corpus/`（題型風格）＋ `statutes/`／即時搜尋（法源）為準；**考古題僅為當年法規快照，不得當現行法引用**。每題須自我驗證答案可在法規中找到。
5. **標準答案**：測驗題於卷末另附標準答案區（`【模擬考】` 標記之內），申論題附答題架構或評分要點，供自我批改；答案區同樣註明為模擬用途。
