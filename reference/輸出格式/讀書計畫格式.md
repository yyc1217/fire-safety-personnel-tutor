# 讀書計畫檔格式範本（study-planner 統一輸出）

> 本檔定義 study-planner 計畫檔（`plans/讀書計畫_<等別><民國年>.md`）之格式，為此輸出的**格式唯一真相**。計畫檔**跨對話讀回解析**（勾銷、重排、進度報告），格式須固定。

---

## 一、版面（兩種形式共用同一檔首）

```markdown
# 讀書計畫（{等別}{民國年}）

> 考試日期：{民國 YYY-MM-DD}（{已確認｜推算值，以考選部公告為準}）｜產出：{YYYY-MM-DD}｜形式：{順序清單｜逐週排程}
```

**（a）順序清單**——`##` 依法規大類分節（標該等別題數；**火災學節**標該等別 `by_topic` 合計題數，位置與單元排序依 `study-planner/SKILL.md` §2 第 5 點），單元行固定三段式「單元（條文）｜建議行動｜對照表」（**無對應內建對照表時第三段仍固定寫 `對照表 —`**，維持三段式欄位、不得省略，以免跨對話解析失敗）：

```markdown
## 第一優先：各類場所消防安全設備設置標準（師 404 題）

- [x] 火警自動警報設備（§19、§112–§128、檢修基準13）｜出題：「考我火警自動警報」｜對照表 04 ✅2026-07-10
- [ ] 自動撒水設備（§17、§43–§60、§211、檢修基準4）｜出題：「考我自動撒水」｜對照表 03
```

**（b）逐週排程**——`##` 依週次分節（附起訖日期），單元行同上；**考前最後 1～2 週固定為衝刺節**：

```markdown
## 第 1 週（7/14–7/20）：自動撒水設備

- [ ] 自動撒水設備（§17、§43–§60、§211、檢修基準4）｜出題：「考我自動撒水」｜對照表 03

## 考前衝刺（考前最後 2 週，自動預留）

- [ ] 弱點總複習：`/弱點複習`
- [ ] 猜題與必背：`/猜題`、`/懶人包`
```

## 二、勾銷與讀回規則

1. 勾銷＝`[ ]` 改 `[x]` 並於**行尾**加 `✅YYYY-MM-DD`；不刪行、不改動單元行三段式結構（機器據以解析）。
2. 重排時**保留已勾銷行**（含日期），只調整未完成行之順序或所屬週次。
3. 啟動時的**進度報告固定格式**（先報告再問接續方式）：

   ```
   讀書計畫進度（{等別}{民國年}）：{x}／{m} 單元完成{，目前第 {w} 週（共 {W} 週）}。
   需加強：{弱點單元，無則省略}。接下來：{下一個未完成單元}。
   要接續勾銷、重排，還是重新產生？
   ```

   - **分母 `m`＝三段式單元行總數**（即含「單元｜建議行動｜對照表」三段的 `- [ ]`／`- [x]` 行），**不含「考前衝刺」節的固定動作項**（`弱點總複習`、`猜題與必背` 等——它們是待辦而非可解析的三段式單元）。分子 `x`＝其中已勾銷（`[x]`）之數。

---

## 三、範例（依本範本產出，逐週排程形式）

# 讀書計畫（師116）

> 考試日期：民國 116-06-05（推算值，以考選部公告為準）｜產出：2026-07-13｜形式：逐週排程

## 第 1 週（7/14–7/20）：火警自動警報設備

- [x] 火警自動警報設備（§19、§112–§128、檢修基準13）｜出題：「考我火警自動警報」｜對照表 04 ✅2026-07-18

## 第 2 週（7/21–7/27）：自動撒水設備

- [ ] 自動撒水設備（§17、§43–§60、§211、檢修基準4）｜出題：「考我自動撒水」｜對照表 03

## 考前衝刺（考前最後 2 週，自動預留）

- [ ] 弱點總複習：`/弱點複習`
- [ ] 猜題與必背：`/猜題`、`/懶人包`

下次執行 `/讀書計畫` 時的進度報告（固定三行）：

```
讀書計畫進度（師116）：1／14 單元完成，目前第 2 週（共 46 週）。
需加強：消防幫浦全揚程計算。接下來：自動撒水設備。
要接續勾銷、重排，還是重新產生？
```
