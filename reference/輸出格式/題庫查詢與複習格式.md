# 題庫查詢與複習模式格式範本（exam-archive／exam-tutor 統一輸出）

> 本檔定義 exam-archive 列清單與查單題、exam-tutor 複習模式之輸出版面，為此類輸出的**格式唯一真相**。

---

## 一、exam-archive：題庫清單

依等別分節（師／士各一表），列＝年度（民國，新在上）、欄＝科目，格號固定：`✓`（有原卷）／`✓答`（含答案卷）／`✓更`（含更正答案）／`—`（無）。

```markdown
### {消防設備師｜消防設備士}

| 年度 | {科目1} | {科目2} | … |
|------|--------|--------|---|
| 115 | ✓ | ✓答 | … |
```

## 二、exam-archive：查單題

首行粗體標卷別與法規快照年，接題目原文（選擇題選項各占一行），末行標準答案；不附 🏷️ 標籤列：

```markdown
**{等別} {年} 年 {科目} 第 {n} 題**（{law_snapshot} 年法規快照）

{題目原文}

標準答案：{X}
```

## 三、exam-tutor：複習模式分節（每設備固定四節）

系統開頭先以一小段講整體脈絡，結尾以「常考數字總整理」收束；每設備：

```markdown
## {設備}

### 設置規定
{設置標準對應條文（含公危場所），條文全文依條項款目}

### 檢修要領
{檢修基準重點}

### 常考數字與公式
{條列附法源}

### 自我檢測
{1–3 題，附答案}
```

## 四、範例（依本範本產出）

| 年度 | 火災學 | 消防法規 | 警報系統 | 避難系統 | 水系統 | 化學系統 |
|------|--------|---------|---------|---------|--------|---------|
| 115 | ✓ | ✓答 | ✓ | ✓ | ✓ | ✓ |
| 114 | ✓ | ✓答 | ✓ | ✓ | ✓ | ✓ |
| 113 | ✓ | ✓答 | ✓ | ✓ | ✓ | ✓ |

查單題：

**師 114 年化學系統第 3 題**（114 年法規快照）

｛題目原文；選擇題選項各占一行｝

標準答案：(B)

```markdown
## 滅火器

｛整體脈絡一小段｝

### 設置規定
｛§14 應設場所、§31 設置規定、公危 §223–§228｝

### 檢修要領
｛檢修基準第 1 章重點｝

### 常考數字與公式
｛條列附法源｝

### 自我檢測
｛1–3 題，附答案｝
```
