# 考前必背懶人包格式範本（statute-memorizer 統一輸出）

> 本檔定義 `/懶人包`（statute-memorizer）的統一輸出格式，為此產出的**格式唯一真相**，每次逐區塊比照。定位：**考前最後衝刺的可背文件**——手機可讀、逐條可背，不是分析報告。

---

## 一、整體結構

```markdown
# 考前必背懶人包（{等別}{民國年}）

> 資料截至 {YYYY-MM-DD}。來源：{猜題範圍（{趨勢報告檔名}）｜依{等別}加權頻率前 {N} 名}＋{平時弱點 {n} 項｜無弱點紀錄}＋內建對照表。
> 條文重點取自本地法規快照，考前請以官方現行版複核。

## {考試科目}

### ⚠ 你的弱點：{考點}
…

### {考點}
…
```

1. **標題含等別與民國年**（＝目標考試年，算法依 `user-config-spec.md`），與存檔名 `考前必背懶人包_<等別><民國年>.md` 一致。
2. **檔首狀態列固定兩行**：來源組成（三來源各標明有無）＋快照免責，之後不再重複。
3. **第一層＝考試科目**（只列本等別、順序比照 `猜題報告格式.md` 第一節）；該科無入選考點則整節省略。

## 二、考點區塊（每考點一個 `###`）

```markdown
### {考點中文名}

- 必背：{數字／關鍵句逐條，一行一個事實}（{法規簡稱 §條號}）
- 必背：{下一條}（{法源}）
- 修法新點：{近期修正內容一句}（{修正日期}；有才列此欄）
- 記憶術：{一句口訣，沿用內建對照表「記憶點」風格}
- 出處：對照表 {NN}（{表名}）（有對應內建表才列）
```

1. **弱點考點一律排該科最前**，標題加前綴 `⚠ 你的弱點：`；其餘考點依猜題範圍／加權頻率排序。
2. `必背：` 每行**只放一個可背的事實**（一個數字組或一句關鍵句），行尾附法源；禁止一行塞多個不相干數字。
3. `修法新點：` 僅在該考點近 12–24 個月確有修正時列出；`出處：` 引內建表須寫「表號＋表名」。
4. **條列不用表格**（背誦文件以逐條為準；使用者要對照全貌時導向 `/對照表`）。
5. 公式一律 LaTeX；數值單位照條文原文。

## 三、結尾（固定一小段）

```markdown
---

弱點來源：{progress.json 弱點紀錄｜你提供的弱點筆記｜無（未記錄）}。
想逐題確認背熟了沒 → `/抽考 {範圍}`；要完整對照表 → `/對照表 {主題}`。
```

## 四、規模與取捨

- 預設**前 20 個考點**（`/懶人包` 參數可改）；弱點考點**不占**此配額、一律納入。
- 每考點 `必背：` 以 3～6 行為度——塞不下表示考點太大，拆成兩個考點；只有 1 行表示太細，併入相近考點。
- 全篇無 emoji（唯一例外：弱點前綴 `⚠`）。

---

## 五、範例（依本範本產出，節錄）

> 節錄：師、水系統科兩個考點。

# 考前必背懶人包（師116）

> 資料截至 2026-07-13。來源：猜題範圍（趨勢報告_師116_2026-07-10）＋平時弱點 2 項＋內建對照表。
> 條文重點取自本地法規快照，考前請以官方現行版複核。

## 水系統消防安全設備

### ⚠ 你的弱點：消防幫浦全揚程計算

- 必背：全揚程 $H = h_1 + h_2 + h_3 + 17\,\text{m}$（落差＋配管摩擦損失＋水帶摩擦損失＋17）（設置標準 §37）
- 必背：代值順序——先分項標明來源（題目給定或條文規定值），最後檢核是否達條文最低值（非法源：計算推導）
- 記憶術：「落差、管損、帶損，最後加十七」
- 出處：對照表 03（水系統設備數值總表）

### 室內消防栓放水性能

- 必背：放水壓力 1.7～7 kgf/cm²（設置標準 §34～§38）
- 必背：放水量第一種 130 L/min 以上、第二種 60 L/min 以上（設置標準 §34～§38）
- 記憶術：「一七到七，一三〇六〇」
- 出處：對照表 03（水系統設備數值總表）
