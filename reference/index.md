# reference/ 內建參考資產索引

本目錄存放 plugin 內建的整理資產（唯讀）。與 `statutes/`（法規全文）、`corpus/`（考古題）的關係：**本目錄是「整理層」**——從 statutes 逐條擷取、供快速取用；法規原文一律以 statutes 與官方現行版為準。

**輸出格式範本集中於 [`輸出格式/`](輸出格式/)**：每種產出一檔＝「格式定義＋範例」同檔（格式唯一真相，各 SKILL 只指向、不重複定義）。

## 使用原則（內建＋即時補充）

1. **內建優先**：使用者要的主題落在上表範圍 → 直接引用對照表（快、且已與 statutes 核對過）。
2. **即時補充**：主題不在內建範圍（如防焰性能、認可基準細節、建築技術規則）→ 即時從 `statutes/` 對應檔逐條整理生成，並標注「即時生成，請核對現行版」；**版面比照內建對照表**（檔首標依據法規與版本日期、欄位附出處條號、易混淆數字附記憶點）。即時生成品質良好且屬高頻主題者，可建議使用者回報作者納入內建。
3. **時效**：每張對照表檔首標注依據法規與版本日期；引用前依各 skill 之「法條時效核對」規則核對官方現行版，發現過期即以官方為準並提示更新。
4. **格式**：數學公式一律 LaTeX（`$...$`／`$$...$$`）；表格用 markdown；易混淆數字附記憶點。

## 維護

- 新增對照表：檔名 `對照表/<序2位>_<主題>.md`，並更新本索引。
- 法規修正時：先更新 `statutes/` 對應檔，再同步修改受影響之對照表與 `設備條文索引.md`。
