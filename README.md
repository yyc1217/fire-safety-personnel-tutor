# fire-safety-personnel-tutor

消防設備師／士國家考試備考 Claude Code plugin。內含民國 100–115 年考古題題庫（160 卷、3680 題，逐題標籤）、命題大綱法規現行全文與高頻考點對照表；所有互動與產出均為繁體中文。

## 功能

| Skill | 職責 |
|-------|------|
| **exam-tutor** | 家教式出題與批改：以消防安全設備為主體、依設備條文索引連續出題（先問後等再解），依實際配分批改（申論每題 0–25 分、選擇題每題 1.25 分）並引用條文全文；含快速抽考、弱點複習、系統複習與申論猜題擬答。 |
| **exam-trend-forecast** | 考前猜題：近 10 年加權考點分布＋**出題週期型態**（常年型換方向重出／每 N 年回鍋／剛考過冷卻中／一次性／未考缺口）＋近 12–24 個月修法／函令／時事，產出**猜題範圍**（範圍＋依據＋型態＋信心程度）。例：瓦斯氣爆重大案件 → 瓦斯漏氣火警自動警報設備設置規定；消防法 §26 平均每 4 年一考、已 10 年未考 → 回鍋候選。只負責猜題，不出題。 |
| **statute-memorizer** | 法規記憶：跨法規對照表（內建 8 張＋即時生成）、記憶卡與口訣、考前必背懶人包（猜題範圍＋平時弱點）。 |
| **study-planner** | 讀書計畫：初次使用先確認考試日期（預設每年六月第一個週末），按師／士命題頻率排研讀順序，進度分配與形式和使用者討論後產出（可勾銷清單或逐週排程，考前自動預留總複習與猜題）；進度看各設備規定是否逐步掌握，落後主動提醒重排，並與出題進度銜接。 |
| **exam-archive** | 題庫查詢：依年度／科目／標籤取原卷 PDF、答案卷或單題原文。 |

### Slash commands

| 指令 | 功能 | 範例 |
|------|------|------|
| `/抽考 [範圍?]` | 快速抽考一輪 3~5 題 | `/抽考 警報系統` |
| `/出考卷 [科目?]` | 套用最新年度原卷版面（首尾標註【模擬考】）產出整卷仿真模擬試卷＋標準答案 | `/出考卷 消防法規` |
| `/弱點複習 [範圍?]` | 依作答紀錄優先重考常錯考點 | `/弱點複習` |
| `/對照表 [主題?]` | 跨法規對照表；留空列出內建清單 | `/對照表 各設備緊急電源` |
| `/猜題 [範圍?]` | 產出猜題範圍（考點＋依據＋週期型態＋信心） | `/猜題` |
| `/申論猜題 [科目?]` | 申論猜題＋答題架構＋擬答（每一論點附法源） | `/申論猜題 消防法規` |
| `/懶人包 [範圍?]` | 猜題範圍＋平時弱點濃縮成必背文件 | `/懶人包` |
| `/讀書計畫 [考試日期?]` | 初次使用先確認考試日期（預設每年六月第一個週末），討論後排研讀順序或週排程 | `/讀書計畫` |
| `/設定` | 應考等別（師／士）、弱點記錄模式、資料目錄 | `/設定` |

## 使用範例

**出題練習**——以消防安全設備為主體、依設備條文索引連續出題，出題後停下等作答；批改時才給分數、條文全文與法源出處。未指定範圍時主動提議（讀書計畫 → 弱點 → 頻率）。以下為六科出題範例（以申論為主）：

火災學（申論）：

```text
"出一題火災學的申論題"
---
閃燃（flashover）與爆燃（backdraft）為建築物火災中最危險的兩種現象：
（一）請分別說明兩者之定義與發生機制。
（二）消防人員抵達火場時，可從哪些徵兆研判即將發生爆燃？
（三）請說明防止爆燃發生之對策。
```

消防法規（申論）：

```text
"考我消防法規"
---
某 12 層樓複合用途建築物內設有補習班與旅館，請依消防法規說明該場所管理權人遴用防火管理人之義務、防火管理人之資格，及其應辦理之事項。
```

水系統（申論）：

```text
"出一題水系統的計算題"
---
一棟 6 層樓辦公大樓，每層設置 2 支第一種室內消防栓。試問：
（一）各層任一點至消防栓接頭之水平距離限制為何？
（二）瞄子放水壓力與放水量之下限各為多少？
（三）水源容量與消防幫浦出水量各應為多少以上？
```

化學系統（申論）：

```text
"考我滅火器"
---
某地面 3 層之補習班，各層樓地板面積均為 400 平方公尺。試問：
（一）該場所是否應設置滅火器？依據為何？
（二）每層所需之最低滅火效能值應如何核算？
（三）滅火器之設置位置有何規定？
```

警報系統（選擇題）：

```text
"出一題警報系統的選擇題"
---
依現行規定，有關火警自動警報設備之火警分區，下列敘述何者錯誤？
(A) 每一火警分區不得超過一樓層，並在樓地板面積 600 平方公尺以下
(B) 上下兩層樓地板面積之和在 500 平方公尺以下者，得二層共用一分區
(C) 每一分區之任一邊長原則上在 50 公尺以下
(D) 裝設光電式分離型探測器時，其邊長得在 150 公尺以下
```

避難系統（選擇題）：

```text
"出一題避難系統的選擇題"
---
緊急照明燈在地面之水平面照度，使用低照度測定用光電管照度計測得之值，在走廊、樓梯及通道等其他場所不得低於多少勒克司（Lux）？
(A) 0.5
(B) 1
(C) 2
(D) 10
```

## 安裝

```bash
claude plugin marketplace add https://github.com/yyc1217/fire-safety-personnel-tutor
claude plugin install fire-safety-personnel-tutor@fire-safety-personnel-tutor-marketplace
```

初次使用任一功能時會詢問應考等別與弱點記錄模式（一次記住，`/設定` 可改）；學習進度存於使用者本機 `~/.fire-safety-tutor/`，plugin 目錄唯讀。

## 目錄結構

```
├── .claude-plugin/          # plugin 與 marketplace 定義
├── corpus/                  # 歷年考古題（md＋原卷 PDF＋標籤索引）
├── statutes/                # 命題大綱法規現行全文 md
├── reference/               # 內建資產：設備條文索引、8 張對照表、使用者設定規格
├── skills/                  # 五個功能 skill＋九個 slash command skill（各為 <名稱>/SKILL.md）
├── scripts/                 # 維護者工具（見 scripts/README.md）
└── docs/                    # 設計筆記、資料維護說明、待辦與里程碑
```

題庫與法規之維護細節（標籤索引流程、新年度試卷納入、法規整理規範）見 [`docs/資料維護.md`](docs/資料維護.md)。

## 設計原則

- **以 corpus 取題型風格，以 statutes／即時搜尋取法源**：考古題僅為當年法規快照，不可當現行法引用。
- **先問、後等、再解**：使用者作答前不給答案。
- **誠實性**：出題前自我驗證、答案須在法規中找得到；猜題附依據與信心程度，不呈現假確定；擬答每一論點附法源。
- **優雅退場**：資料缺漏或無網路時降級運作並說明，不可杜撰條文。
