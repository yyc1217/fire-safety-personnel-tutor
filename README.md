# fire-safety-personnel-tutor

消防設備師／士國家考試備考 Claude Code plugin。內含民國 100–115 年考古題題庫（160 卷、3680 題，逐題標籤）、命題大綱法規現行全文與高頻考點對照表；所有互動與產出均為繁體中文。

## 功能

| Skill | 職責 |
|-------|------|
| **exam-tutor** | 家教式出題與批改：以消防安全設備為主體、依設備條文索引連續出題（先問後等再解），0–100 分批改並引用條文全文；含快速抽考、弱點複習、系統複習與申論猜題擬答。 |
| **exam-trend-forecast** | 考前猜題：近 10 年加權考點分布＋近 12–24 個月修法／函令／時事，產出**猜題範圍**（範圍＋依據＋信心程度）。例：瓦斯氣爆重大案件 → 瓦斯漏氣火警自動警報設備設置規定。只負責猜題，不出題。 |
| **statute-memorizer** | 法規記憶：跨法規對照表（內建 8 張＋即時生成）、記憶卡與口訣、考前必背懶人包（猜題範圍＋平時弱點）。 |
| **study-planner** | 讀書計畫：按師／士命題頻率排研讀順序，產出可勾銷清單或逐週排程（考前自動預留總複習與猜題），並與出題進度銜接。 |
| **exam-archive** | 題庫查詢：依年度／科目／標籤取原卷 PDF、答案卷或單題原文。 |

### Slash commands

| 指令 | 功能 | 範例 |
|------|------|------|
| `/抽考 [範圍]` | 快速抽考一輪 3~5 題 | `/抽考 警報系統` |
| `/弱點複習 [範圍]` | 依作答紀錄優先重考常錯考點 | `/弱點複習` |
| `/對照表 [主題]` | 跨法規對照表；留空列出內建清單 | `/對照表 各設備緊急電源` |
| `/猜題 [範圍]` | 產出猜題範圍（考點＋依據＋信心） | `/猜題` |
| `/申論猜題 [科目]` | 申論猜題＋答題架構＋擬答（每一論點附法源） | `/申論猜題 消防法規` |
| `/懶人包 [範圍]` | 猜題範圍＋平時弱點濃縮成必背文件 | `/懶人包` |
| `/讀書計畫 [日期 時數]` | 排研讀順序；附考試日期與週時數則產週排程 | `/讀書計畫 115-06-14 每週10小時` |
| `/設定` | 應考等別（師／士）、弱點記錄模式、資料目錄 | `/設定` |

## 使用範例

**出題練習**——依設備條文索引連續出題（應設場所 → 設置規定 → 公危 → 檢修基準 → 延伸知識），出題後停下等作答；未指定設備時主動提議（讀書計畫 → 弱點 → 頻率）。

```text
你：     考我滅火器
Claude： 〔滅火器 1/9・設置標準 §14〕考古題：下列何種場所，不論面積大小
         均應設置滅火器？(A)…(B)…(C)…(D)…
你：     (C)
Claude： ✅ 100 分。依《各類場所消防安全設備設置標準》第 14 條第 4 款（113-04-24
         修正版）：「地下建築物、幼兒園……」〔條文全文〕
         下一題〔滅火器 2/9・§31 設置規定〕：……
```

**系統複習**——整批講解該系統條文、設備、檢修要領與常考數字，講解為主、測驗為輔。

```text
你：     帶我看水系統的規定
Claude： 水系統複習（依近五年頻率排序）：一、自動撒水設備——應設場所 §17、
         撒水頭配置 §46～§50（放水量 80 L/min、壓力 1 kgf/cm²）……
         〔段落結束〕小測驗：高架倉庫撒水頭放水量是多少？
```

**考前猜題**——`/猜題` 只產出「猜題範圍」（範圍＋依據＋信心）；練題接 `/申論猜題`，背重點接 `/懶人包`。

```text
你：     /猜題
Claude： 【時事預測｜信心：中】去年○○瓦斯氣爆案 → 建議優先準備瓦斯漏氣火警
         自動警報設備設置規定與檢知器裝設（設置標準 §21、§140～§145-1）
         【法規脈動｜信心：高】消防法 §13 修正 → 施工中消防防護計畫（含修法理由）
         ……（資料截至：2026-07）想練題 → /申論猜題；想背重點 → /懶人包
```

**申論擬答**——`/申論猜題`：預測題目＋答題架構＋擬答，每一論點附法源，無法源之推論標注「非法源」。

```text
你：     /申論猜題 消防法規
Claude： 預測題：試述施工中消防防護計畫之訂定時機與應辦事項。（25 分）
         答題架構：破題（§13 修法背景）→ 法源 → 分項論述 → 結論
         擬答：依《消防法》第 13 條第 3 項（113-11-29 修正版）……
         ※ 非法源：實務見解——實務上多於申報前預留改善期間。
```

**考前衝刺**——`/懶人包`：猜題範圍＋平時弱點濃縮成必背文件；`/弱點複習`：重考錯過的題目直到克服。

```text
你：     /懶人包
Claude： 考前必背（師 115）：
         1. ⚠ 你的弱點｜滅火器防護距離——§224 第 4 種 ≤30 m、§225 第 5 種 ≤20 m
            記憶術：「四三、五二」
         2. 瓦斯漏氣檢知器——輕 8 重 4、輕上重下（對照表 04）
         ……存入 forecasts/考前必背懶人包_師115.md？
```

**規劃進度**——`/讀書計畫`：依命題頻率排單元；附考試日期與週時數則產逐週排程，考前 1~2 週自動預留總複習與猜題。

```text
你：     /讀書計畫 115-06-14 每週10小時
Claude： 距考試 23 週（扣考前 2 週總複習）。第 1 週：火警自動警報設備
         （§19、§112～§128、檢修基準 13）約 6 小時｜行動：「考我火警自動警報」
         ＋ 對照表 04……第 22～23 週：/弱點複習＋/猜題＋/懶人包
```

**查卷**——自然語言查題庫。

```text
你：     給我 113 年設備師消防法規的原卷
Claude： corpus/pdf/師/113/0802_消防法規.pdf（md 全文如下）……
你：     列出考過緩降機的題目
Claude： 以標籤 by_equipment:避難器具 取題後篩「緩降機」，列出題目參照
         （等別/年/科目#題號）與題幹摘要……
```

## 安裝

```bash
claude plugin marketplace add https://github.com/yyc1217/fire-safety-personnel-tutor
claude plugin install fire-safety-personnel-tutor@fire-safety-personnel-tutor-marketplace
```

初次使用任一功能時會詢問應考等別與弱點記錄模式（一次記住，`/設定` 可改）；學習進度存於使用者本機 `~/.fire-safety-tutor/`，plugin 目錄唯讀。

## 目錄結構

```
├── .claude-plugin/          # plugin 與 marketplace 定義
├── corpus/                  # 歷年考古題（md＋原卷 PDF＋標籤索引）
├── statutes/                # 命題大綱法規現行全文 md
├── reference/               # 內建資產：設備條文索引、8 張對照表、使用者設定規格
├── commands/                # slash commands
├── skills/                  # 五個 skill 之 SKILL.md
├── scripts/                 # 維護者工具（見 scripts/README.md）
└── docs/                    # 設計筆記、資料維護說明、待辦與里程碑
```

題庫與法規之維護細節（標籤索引流程、新年度試卷納入、法規整理規範）見 [`docs/資料維護.md`](docs/資料維護.md)。

## 設計原則

- **以 corpus 取題型風格，以 statutes／即時搜尋取法源**：考古題僅為當年法規快照，不可當現行法引用。
- **先問、後等、再解**：使用者作答前不給答案。
- **誠實性**：出題前自我驗證、答案須在法規中找得到；猜題附依據與信心程度，不呈現假確定；擬答每一論點附法源。
- **優雅退場**：資料缺漏或無網路時降級運作並說明，不可杜撰條文。
