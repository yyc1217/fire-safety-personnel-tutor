# 法規版本追蹤

> 目的：`statutes/` 各 md 為版本快照，官方隨時可能修正。本檔記錄**各檔應追蹤的官方沿革來源**、
> **已核對之官方最新版本**與**已知過期清單**，並由 `scripts/check_statute_versions.py` 機械比對。
> 背景見 issue #31（檢修基準快照過期 18 個月才被發現）。

## 一、使用方式

```bash
python3 scripts/check_statute_versions.py        # 列出全部檔案之本地版本日期與核對狀態
python3 scripts/check_statute_versions.py --stale-only   # 只列 🔴 過期與 ⚠️ 無法解析者
```

- 腳本讀各檔**檔首版本日期**，與本檔第三節「核對紀錄表」之「官方最新已知版本」比對：
  本地較舊 → 🔴 過期；相同 → ✅ 現行（以核對日期為準）；表中無紀錄 → ⚪ 未核對。
- **核對節奏**（沿用 `statutes/index.md`「法規時效」第 4 點）：至少每季、或每次報名考試後全查一次；
  重大修法公告後立即核對。核對後把結果寫回第三節之表（含核對日期），過期者同步在該檔檔首加
  「🔴 已知過期」警示（格式見 `statutes/2_03` 檔首）。

## 二、官方沿革來源與可達性

| 來源 | 網址 | 用途 | 可達性備註 |
|------|------|------|------|
| 全國法規資料庫 | <https://law.moj.gov.tw/> | 法律、法規命令之現行版與沿革（如設置標準 pcode=D0120029） | 正常 |
| 內政部主管法規共用系統 | <https://glrs.moi.gov.tw/> | 內政部發布之行政規則沿革 | 正常 |
| 行政院公報資訊網 | <https://gazette.nat.gov.tw/> | 修正令原文（含逐點修正對照） | 正常（實測可達） |
| 內政部消防署 | <https://www.nfa.gov.tw/>、<https://law.nfa.gov.tw/> | 認可基準、檢修基準等消防署行政規則 | ⚠️ **憑證鏈不完整**（`unable to verify the first certificate`），程式化抓取（WebFetch／curl）常失敗；不可達時改用左列三來源，或以 web 搜尋「法規名＋修正」取公報／主管法規系統頁面 |
