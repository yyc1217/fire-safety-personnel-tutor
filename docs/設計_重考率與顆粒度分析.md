# 設計：考點重考率 × 顆粒度分析

> 對應工具：`scripts/analyze_repeat_granularity.py`＋詞庫 `scripts/repeat_lexicon.json`
> 產物：`corpus/命題重考率分析.md`（人可讀）、`corpus/repeat_granularity.json`（機器可讀）
> 定位：**探索性分析**，回答「前一年考過的考點今年不會再考」是否可信。

## 1. 動機

坊間備考口訣常稱「前一年考過的考點今年不會再考」。若為真，可用來篩題省力；若為假，
照做會漏掉年年必考的核心。本工具以民國 100–115 年題庫實測其可信度，並**分等級（師／士）
與考科**檢視。

## 2. 核心觀念：重考率必須對照「隨機基準」

單看「重考率低」不能證明命題者刻意迴避去年考點。原因：把考點切得越細、桶（類別）越多，
而每年題數固定，任一特定細考點本來就**大概率不會剛好連兩年中**——這是純機率，不是迴避。

因此本工具的關鍵不是重考率本身，而是**觀測重考率 vs 隨機基準**：

- **重考率**：連續兩年 Y-1→Y，前一年考點集合中今年又出現者之比例（分等級／分層計算，
  僅計真正連續之年份）。
- **隨機基準（置換檢定）**：固定每年題數與桶標籤多重集合，將桶標籤在題目間隨機重排
  N 次（預設 3000），重算重考率，得到「命題與年份無關」時的分布與 95% 區間。
- **判讀**：
  - 觀測值落在隨機分布內（百分位 5–95%）→ **與隨機無異**，重考率高低純屬顆粒度機率，
    命題者**未**刻意迴避去年考點。
  - 觀測值顯著低於隨機（百分位 <2.5%）→ 疑有**迴避傾向**（仍須檢視樣本量）。
  - 觀測值顯著高於隨機（>95%）→ 比隨機**更常**重考（核心考點反覆出）。

## 3. 四層顆粒度

| 層 | 定義 | 詞庫來源 |
|---|---|---|
| 設備 | 室內消防栓／自動撒水／泡沫／乾粉／海龍… | 檢修基準各章名＋設置標準 |
| 組件 | 加壓送水裝置、一齊開放閥、末端查驗閥、儲存容器… | 檢修基準各章「檢修項目」`（N）組件` 次標題 |
| 子考點 | 撒水頭種類／放水量／水源容量計算／啟動裝置… | 人工設計（焦點設備） |
| 子子考點 | 標示溫度／玻璃球強度試驗／耐洩漏試驗… | 人工設計（焦點桶，示範用） |

- 設備／組件層對整卷題目分類（未閘門）；子考點／子子考點層以 `_gate` 關鍵字**限定焦點設備**
  （水系統焦點＝自動撒水設備；化學系統焦點＝氣體滅火設備通用）。
- 每題只取**題幹**（第一個選項 `- (A)` 之前）比對，避免選項附帶提及灌水；有序詞庫取第一個
  命中桶為主考點。

## 3-1. 報告編排（各等別獨立）

`corpus/命題重考率分析.md` **先分師／士、再分系統**：師篇涵蓋全部系統之階梯與桶分布，末附師篇小結與實務建議（申論版）；士篇同（測驗版）；最後為兩等別共通裁決。桶分布與未分類率亦分等別。合併（跨等別）數據不作主要判讀，僅留存於 `repeat_granularity.json`。此編排落實「師卷與士卷命題委員不同、各等別獨立」之鐵則。

## 4. 主要發現（詳見 `corpus/命題重考率分析.md`）

- **設備／組件層**：核心設備與主力組件（撒水頭、消防幫浦、藥劑儲存容器…）幾乎**年年回鍋**，
  重考率 70–90%，且**與隨機基準無異**。→ 此命題在粗顆粒度明顯不可信。
- **子考點層**：重考率降到 45–62%，但（樣本足夠時）**仍與隨機無異**。→ 下降是顆粒度機率，
  非迴避。
- **子子考點層**：重考率再降，個別出現「顯著低於隨機」，但樣本極小（單設備單桶、數十題）
  且屬事後多重比較，多為雜訊，**僅示範顆粒度效應，不作實證結論**。

**裁決**：命題「前一年考過的考點今年不會再考」——**當篩題規則不可信**；僅在「同一核心組件
今年多半換一個子考點角度切入」這層意義上近似成立，是「換角度」而非「不考」。

## 5. 界線與誠實警語（務必遵守）

1. **不回寫語意標籤**：子考點／子子考點採**關鍵字啟發式**分桶，準確度有限——此即
   `scripts/build_tags.py` 於 2026-06 停用之教訓（關鍵字自動判讀會覆蓋語意標籤）。
   本工具產出僅為分析報告，**絕不寫入** `corpus/tags_index.json` 或 inline `🏷️` 標籤；
   語意標籤仍為唯一真相來源。
2. **分桶粗細＝人為設定**：桶的切法直接左右重考率，報告須附詞庫供檢視；改詞庫須重跑並
   人工抽樣核對分類。
3. **未分類率**：已閘門層（子考點／子子考點）未分類率 >20% 標 🔴 表詞庫待補；未閘門層
   （設備／組件）之分母含他系統題（士為水化學合併卷）與跨設備綜合題，未落桶非必為缺口。
4. **樣本量**：師卷每年僅 4 題申論，細層樣本極小，數字僅供參考。
5. 一切以官方現行版與語意標籤為準；本報告為統計啟發式，非命題保證。

## 6. 使用與擴充

```bash
python3 scripts/analyze_repeat_granularity.py            # 產出報告與 JSON
python3 scripts/analyze_repeat_granularity.py --perm 5000  # 調整置換次數
```

- 現涵蓋**水系統、化學系統、警報系統、避難系統、公共危險物品場所**（末者為法規／場所領域，各層均加 hazmat `_gate`，層名沿用框架但語意為場所大類→主題→焦點）。擴充其他系統：於 `scripts/repeat_lexicon.json` 之 `systems`
  下新增鍵（`papers_glob`＋`layers`），毋需改程式。
- 子子考點層目前僅為水系統自動撒水設備之「撒水頭桶」示範；如需其他焦點桶，比照新增
  `_focus`／`_gate`／`_gate_exclude` 控制鍵即可。
