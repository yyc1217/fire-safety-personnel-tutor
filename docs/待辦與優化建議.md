## 1. 題目查詢複雜度：現況分析與改善方案

### 現況（不是 O(1)）

目前的存取流程分兩步：

1. **O(1)** — 從 `corpus/tags_index.json` 取得題目參照（例如 `士/112/0807#Q15`），
   推算出檔案路徑 `corpus/md/士/112/0807_火災學概要.md`。
2. **O(k)** — 逐行讀取該 markdown 檔直到找到 `### 第 15 題` 標題。
   實測：一份試卷 md 約 14 KB，Q15 位於第 227 行，skill 必須線性掃過前 226 行。

整體複雜度 = O(1) 檔案定位 + O(k) 檔案內掃描，**k = 題號 × 每題平均行數**。

### 改善方案：建立 `corpus/questions.json` 平坦索引

新增一支腳本 `scripts/build_questions_index.py`，在 `build_tags.py` 之後執行，
產出 `corpus/questions.json`，結構如下：

```json
{
  "士/112/0807#Q15": {
    "ref":    "士/112/0807#Q15",
    "type":   "測驗題",
    "stem":   "下列有關火羽流的敘述，何者正確？",
    "options": {
      "A": "...",
      "B": "...",
      "C": "...",
      "D": "..."
    },
    "answer": "C",
    "tags":   ["測驗題", "火災學", "火羽流", "找正確"],
    "md":     "corpus/md/士/112/0807_火災學概要.md"
  },
  "士/112/0807#甲1": {
    "ref":    "士/112/0807#甲1",
    "type":   "申論題",
    "stem":   "...",
    "answer": null,
    "tags":   ["申論題", "火災學", "燃燒理論"],
    "md":     "corpus/md/士/112/0807_火災學概要.md"
  }
}
```

Skill 只需 `questions_index[ref]` 即可取得完整題目，真正達到 **O(1)**。

### 注意：markdown 仍是唯一真相來源

`questions.json` 為衍生產物（derived artifact），應：
- 列入 `.gitignore`，不進版控（或標註 auto-generated，不手動修改）。
- 在 CI / pre-push hook 或 skill 初次載入時自動重建。
- 若 markdown 被修改，必須重新執行 `build_questions_index.py`。

## 2. 標籤人工校正（自動分類準確度）

`build_tags.py` 用關鍵字比對自動分類系統、設備、法規、知識領域，
設計文件已標注「建議人工校正」。已知潛在問題：

- **系統歸屬模糊**：「緊急廣播」同時涉及警報系統與避難系統，目前只取第一匹配。
- **法規 by_law**：僅依題幹關鍵字，對引用多法條的申論題可能漏標。
- **跨科題目**：例如 0808（消防法規）題目也可能包含滅火設備計算，現僅標 `法規`，
  缺少設備標籤。

建議校正優先順序：by_equipment → by_law → by_system。

目前 exam-tutor 和 exam-trend-forecast 技能尚未明確宣告
以 `tags_index.json`（或未來的 `questions.json`）作為查詢入口。

應更新各 SKILL.md，加入：
```
查詢流程：
1. 讀 corpus/questions.json（O(1) 題目存取）
2. 依 tags_index.json 的 by_system / by_equipment / by_flag 篩選出題範圍
3. 不需逐一掃描 corpus/md/** 目錄
```

## 4. 法規附表附圖驗證（Standing Instruction）

每次上傳完一批法規後，**逐一核對** `statutes/附件/` 下的附表與圖片是否正確轉換：

- 已內嵌 markdown 的表格：目視確認欄位、數字是否與原始 PDF/圖片吻合
- 僅附連結的複雜表格：點開原始檔，確認 AI 拆解後的結構描述是否正確
- 標記「（見原檔）」的儲存格：表示 AI 無法確認其值，需人工補入

相關設定亦記載於根目錄 `CLAUDE.md`（Claude Code 每次 session 自動讀取）。
