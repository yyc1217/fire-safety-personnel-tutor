## 一、題目存取效能：`corpus/questions.json` 平坦索引（提案，未實作）

現況存取分兩步：`tags_index.json` 取得參照（O(1)）→ 逐行掃描該 md 找 `### 第 N 題`（O(k)）。

**改善**：新增 `scripts/build_questions_index.py`（於 `build_tags.py` 後執行），產出 `corpus/questions.json`，每題存 `{ref, type, stem, options, answer, tags, md}`，skill 以 `questions_index[ref]` 達到真正 O(1)。

注意：markdown 為唯一真相來源；`questions.json` 為衍生產物，應列入 `.gitignore` 或標註 auto-generated，md 變更後須重建。

## 二、標籤分類校正（以語意判斷為準，非關鍵字 script）

**方針（2026-06-19 起）**：分類準確度以**語意判斷**為準，不再以 `build_tags.py` 關鍵字結果為最終值。`build_tags.py` 僅作初稿與「題型／旗標／條號」等客觀維度之來源；**系統／設備／法規／知識領域以各題 inline `🏷️` 標籤為唯一真相來源**。校正流程：直接編輯該題 `🏷️` 行 → 執行 `scripts/rebuild_index.py`（解析 inline 標籤反建 `tags_index.json`，不重判題目）。

**已釐清**：緊急廣播設備於考試範圍**屬警報系統（警報設備）**，依此分類（曾一度改列避難系統，經使用者確認後已還原）。

**仍待語意校正（使用者人工核對中，本項保留）**：
- **師設備科目申論（重要）**：警報／避難／水／化學系統 64 卷為全申論卷（原中文數字標題未被舊正則辨識，已補正並 100% 補標）。其中約 50 卷 200 題目前為**確定性基線標籤**（申論題｜系統｜計算題），**法規／設備維度尚待語意精修**（代理多次觸限未竟，俟限速重置以子代理或人工補 `各類場所消防安全設備設置標準` 等具體法規＋設備＋條號）。
- 跨系統題：含多設備者之系統歸屬是否齊全。
- by_law 漏標：引用多法條之申論題可能僅標到部分。
- 跨科題：法規卷中含設備計算者，可能缺設備標籤。
- 設備過度標記：關鍵字誤命中（如題幹僅順帶提及）者應移除。

校正優先序：師設備申論法規/設備精修 → by_equipment → by_system → by_law。
