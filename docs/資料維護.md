# 資料維護（題庫與法規）

> 本文件為維護者說明（自 README 移入），涵蓋題庫現況、標籤索引維護與資料準備流程。

## 題庫現況

- **設備師（師）**：民國 100–115 年六科；**設備士（士）**：100–115 年四科，共 **160 卷**，原卷 PDF 與 md 均全數入庫，分置 `corpus/pdf/` 與 `corpus/md/`。
- 全庫共 **3680 題**，已逐題標籤（題型／系統／設備／法規／條號／知識領域／旗標）。
- **各題 inline `🏷️` 標籤為唯一真相來源**：校正分類請直接編輯該題 `🏷️` 行，再執行 `scripts/rebuild_index.py` 反建 `corpus/tags_index.json`，並以 `scripts/analyze_corpus.py` 重產 `corpus/tags_summary.json` 與 `corpus/命題頻率分析.md`、`scripts/analyze_cycles.py` 重產 `corpus/tags_cycles.json` 與 `corpus/命題週期分析.md`。
- 標籤維度定義見 `docs/設計_題目標籤系統.md`；各腳本用途見 [`scripts/README.md`](../scripts/README.md)。

## 法規現況

命題大綱所列法規（`statutes/index.md` 第一～四節）已全數整理為現行全文 md，並完成人工核對；各檔檔首附「📌 免責聲明」與版本日期，引用前請依官方公告核對時效。

## 資料準備流程

1. **考古題**：新增年度或士類試卷時，依 [`corpus/INGEST.md`](../corpus/INGEST.md) 自考選部下載官方 PDF、轉成 md、更新 `corpus/index.json`。
2. **法條**：依 [`statutes/index.md`](../statutes/index.md) 的清單與格式規範，自全國法規資料庫／內政部消防署整理現行全文 md 放入 `statutes/`（檔名規則與附表處理原則見該索引及專案 CLAUDE.md）。

## 使用者資料邊界

skill 對 `corpus/`、`statutes/`、`reference/` 一律唯讀。使用者設定與學習進度存於使用者本機（預設 `~/.fire-safety-tutor/`，plugin 之外），初次使用詢問一次並記住；規格單一來源見 [`reference/user-config-spec.md`](../reference/user-config-spec.md)。
