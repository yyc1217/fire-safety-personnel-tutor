# corpus 納入程序（INGEST）

本文件說明如何將新年度的消防設備師／士考古題納入題庫。此流程為**人工作業**：考選部檔案端點會阻擋程式直連，且考試代碼逐年重編、無公式可推（例如 114 年消防設備人員為 114050，而 114060 是同年警察特考），故不可自動化。

## 步驟

1. **取得官方 PDF**：於[考選部考畢試題查詢平臺](https://wwwq.moex.gov.tw/exam/wFrmExamQandASearch.aspx)手動下載該年度消防設備師（六科）／消防設備士（各科）的試題 PDF（含測驗式試題答案卷）。
2. **轉成 md**：自行將每份試卷的**文字**轉為 md（格式見下），**原始 PDF 一併放入同目錄**（供 AI 查看題目附圖）。
3. **更新 `index.json`**：為每份試卷新增一筆 `papers` 紀錄（欄位見下），並更新 `years`。

## 檔案放置與命名

```
corpus/
├── md/                                   # AI 讀取用之轉換文字（純 md）
│   ├── 師/<民國年>/<科目代碼>_<科目名稱>.md
│   └── 士/<民國年>/<科目代碼>_<科目名稱>.md
└── pdf/                                  # 原始 PDF（原卷、答案卷、更正答案卷）
    ├── 師/<民國年>/<科目代碼>_<科目名稱>.pdf
    └── 士/<民國年>/<科目代碼>_<科目名稱>.pdf
```

> **md 與 pdf 分屬不同上層資料夾**（便於分別下載：只要文字就抓 `md/`，要原卷就抓 `pdf/`）。兩者在各自樹下維持相同的「等別／年／檔名」階層，靠 index.json 的 `md`、`pdf` 欄位對應，不需同目錄。

例：`md/師/114/0801_火災學.md` ＋ `pdf/師/114/0801_火災學.pdf`

含測驗題之科目另有官方「測驗式試題標準答案」PDF，命名為 `<科目代碼>_<科目名稱>_答案.pdf` 放入 `pdf/` 對應目錄，並在 index.json 以 `ans_pdf` 欄位登錄。

> 士為「概要」版科目（普通考試），其卷與師同為「申論 2 題＋測驗 40 題」混合題型；科目分四科：火災學概要、消防法規概要、警報與避難系統消防安全設備概要、水與化學系統消防安全設備概要，科目代碼依實際試卷為準。

## md 內容格式

- 檔首列出考試名稱、等別、類科、科目、試卷代號、原卷檔名與法規快照警語（範例見 `md/師/114/0802_消防法規.md`）。
- 保留原卷題號與配分；申論、測驗分節。
- 測驗題選項依原卷以 `(A)(B)(C)(D)` 標示，每題之後以 `**標準答案：(X)**` 標注官方答案（取自答案卷 PDF；有更正答案卷者以更正為準）。
- **圖的處理**：md 內凡有圖處，標一行指標指向原卷 PDF 頁次，例：

  ```
  > 〔圖：見原卷 pdf/師/114/0801_火災學.pdf 第 1 頁，中性帶示意圖〕
  ```

  skill 遇到需要圖的題目時，會依此開啟該 PDF 指定頁查看（以 `pdftoppm` 轉圖再判讀）。

## index.json 欄位說明

```json
{
  "levels": ["師", "士"],
  "years": ["114", "113"],
  "papers": [
    {
      "level": "師",
      "year": "114",
      "subject": "0801",
      "name": "火災學",
      "md": "師/114/0801_火災學.md",
      "pdf": "師/114/0801_火災學.pdf",
      "exam_code": "114050",
      "paper_code": "40110",
      "types": ["申論"],
      "q_count": 4,
      "has_figure": true,
      "law_snapshot": "114年6月"
    }
  ]
}
```

| 欄位 | 說明 |
|------|------|
| `level` | 「師」或「士」 |
| `year` | 民國年 |
| `subject` | 科目代碼（依原卷） |
| `name` | 科目名稱 |
| `md` / `pdf` | 相對 `corpus/` 的路徑 |
| `ans_pdf` | （選填）官方測驗式試題標準答案 PDF 路徑，僅含測驗題之科目需要 |
| `mod_ans_pdf` | （選填）官方「測驗題標準答案更正」PDF 路徑（命名 `_更正答案.pdf`）；存在時 md 內答案以更正後為準 |
| `exam_code` | 考選部該年度考試代碼 |
| `paper_code` | 試卷代碼 |
| `test_code` | （選填）測驗式試題代號，僅含測驗題之科目需要 |
| `types` | 題型，如 `["申論"]`、`["測驗"]`、`["申論","測驗"]` |
| `q_count` | 總題數 |
| `q_detail` | （選填）各題型題數，如 `{"申論": 2, "測驗": 40}` |
| `has_figure` | 是否含圖（skill 據此決定是否翻 PDF） |
| `law_snapshot` | **重要護欄**：此卷反映「當年」的法規狀態，**不可當現行法引用**，僅供題型風格參考 |
