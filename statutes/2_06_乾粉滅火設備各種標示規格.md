一、本規格依各類場所消防安全設備設置標準第九十八條規定訂定之。

二、乾粉滅火設備使用之各種標示規格應符合下列規定：

（一）手動啟動裝置標示規格如下：

1. 尺寸 A：300 mm 以上；B：100 mm 以上。

2. 紅底白字。

標示牌文字（A 為長、B 為高）：

```
┌─────────────────┐
│  乾粉滅火設備      │  B
│  手動啟動裝置      │
└─────────────────┘
        A
```

（二）放射表示燈規格如下：

1. 尺寸 A：280 mm 以上；B：80 mm 以上。

2. 字體大小：第一行字長、寬為 35 mm 以上；第二行字長、寬為 25 mm 以上。

3. 平時底及字樣均為白色。

4. 點燈時白底紅字。

5. 燈具本體為紅色。

標示牌文字：

```
┌─────────────────┐
│  乾粉放射中       │  B
│  危險禁止進入      │
└─────────────────┘
        A
```

（三）移動放射方式標示規格如下：

1. 尺寸 A：300 mm 以上；B：100 mm 以上。

2. 紅底白字。

標示牌文字：

```
┌─────────────────┐
│  移動式乾粉滅火設備    │  B
└─────────────────┘
        A
```

（四）音響警報裝置標示規格如下，須設於室內明顯之處所：

1. 尺寸 A：480 mm 以上；B：270 mm 以上。

2. 黃底黑字。

3. 每字大小為 25 mm × 25 mm 以上。

標示牌文字：

```
┌──────────────────────────┐
│  室內設有乾粉式滅火設備          │
│  放射前警鈴響時                │  B
│  請立即退避室外                │
└──────────────────────────┘
              A
```
